Internet protocol version four is fourth version of the internet protocol and currently used version of IP address. It’s the type of protocol which is based on standard-base internetworking method in the internet. It’s connectionless protocol which is used on packet-switching networks. It uses 32 bits numeric address which is printed as four numbers alienated by dots and has limitation address space up to 232 addresses. It consists of 4 octets of the address which is spoken individually in decimal numbers and each octet is separated by dots. The total number of IPv4 addresses ranges from 000.000.000.000 to 255.255.255.255. IPv4 reserves special address block for private network and multicast address which seems like large in number but it’s not enough for the entire device connected to internet world.
